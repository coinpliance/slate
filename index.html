    Merchants

    Create new Merchant

    curl -H "Content-type: application/json" --data-binary @$1 -u <companyname>api:(apiKey) -XPUT http://<companyname>:8501/<companyname>/>apikey> -XPUT http://coinpliance.us/<companyname>/api/merchants/<companyname>db:$2

    —————- Returns——————

    {
  "reasonCode": code,
  "upgradeLink": "Some upgrade https link"
}

    This endpoint sends a tier upgrade request to Coinpliance.

    PUT Request

    PUT https://coinpliance.us/<companyname>/merchants/<ID>

    Body Params

    Parameter Description
    appliedTier the tier the merchant would like to go to

    Reason Codes

    Code Description
    1010 Holy crap something blew up. Abort
    1011 No docs required for this upgrade
    1012 Docs already submitted for this possible upgrade
    1013 Invalid request to upgrade to a lower tier
    1009 Success
